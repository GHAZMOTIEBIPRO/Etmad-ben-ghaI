# حالة مصدر بيانات اعتماد

## قرار الـMVP

- المصدر الافتراضي: `DATA_SOURCE=mock`.
- يوجد Connector مستقل باسم `EtimadOpenDataConnector`.
- لا يوجد Web Scraping أو تجاوز حماية أو تسجيل دخول أو Captcha أو Rate Limits.
- عند اعتماد API رسمي أو ملف بيانات مفتوحة حكومي موثق، يتم تعديل الـConnector والـmapping فقط.

## متطلبات التفعيل الحقيقي

1. التحقق من أن المصدر رسمي وعام أو أن الحساب مخول باستخدامه.
2. مراجعة شروط الاستخدام وحدود المعدل.
3. توثيق Schema المصدر.
4. بناء mapping صريح إلى `Tender` بدل casting المؤقت.
5. اختبار Idempotency والتكرار قبل تشغيل Cron الإنتاجي.

## مراجع رسمية للمتابعة

- بوابة منصة اعتماد: `https://portal.etimad.sa/`
- عند توفر مصدر رسمي، يوثق رابطه وتاريخ التحقق وشروط الاستخدام هنا قبل تفعيله إنتاجيًا.
