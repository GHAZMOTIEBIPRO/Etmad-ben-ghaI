# FURAS integration

The Construction Radar uses the public FURAS investment-opportunity listing as a live source. The connector only reads fields visible to anonymous visitors, such as opportunity reference, public title/context, region hints, brochure price, and submission deadline when present.

It does not sign in, purchase RFP documents, submit bids, or access investor-only data.

The authenticated FURAS workflow remains external to the radar and should only be integrated through an official authorized interface if one becomes available.
